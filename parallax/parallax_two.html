Custom Parallax
A customizable, modular, responsive, parallax plugin for jQuery. How use:
1) Get js code and add to your project.
2) Select type of parallax - Vertical/Horizontal and add data-attr to parent block. Examplex - data-parallax-type='vertical'.
3) Add class 'c-rellax' to your image.
4) Add data-attr with speed - data-parallax-speedY='1'. But, if you use Horizontal type, you show also add second data-attr speed - data-parallax-speedX='1'.
